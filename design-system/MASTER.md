# Mirana Crypto Dashboard - Design System

## Product & Style
- **Product**: Fintech / Crypto Dashboard
- **Style**: Minimal, professional, clean
- **Industry**: Cryptocurrency, trading

## Color Palette (from Figma)
- **Primary**: #E25706 (orange), #BA420D (dark orange)
- **Secondary**: #7A4DF5 (purple), #F8F5FF (light purple)
- **Greyscale**: #09090B, #3F3F46, #52525B, #71717A, #D4D4D8, #E4E4E7, #F4F4F5, #FAFAFA, #FFFFFF
- **Success**: #1FB620, #1A981B, #EDFEED
- **Error**: #B21634, #96132C, #FCE8EC

## Typography
- **Font**: Public Sans (Google Fonts) - weights 400, 500, 600, 700
- **Line height**: 1.5-1.75 for body
- **Letter spacing**: -0.28px to -0.64px per size

## Spacing & Radius
- **Border radius**: 6px (small), 8px (buttons), 12px (icon buttons, cards), 16px (large cards)
- **Shadow XS**: 0 1px 2px rgba(13,13,18,0.06)

## UX Rules (from UI/UX Pro Max)
- Use SVG icons (Lucide), never emojis
- cursor-pointer on all clickable elements
- Minimum 44x44px touch targets
- Visible focus rings for keyboard nav
- 150-300ms transitions for micro-interactions
- 4.5:1 color contrast for text

## Anti-Patterns to Avoid
- No emoji icons
- No layout shift on hover (avoid scale transforms)
- Sufficient contrast in light mode
